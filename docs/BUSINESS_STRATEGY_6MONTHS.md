## 💰 商业模式设计

### 1. 定价策略

#### 三层定价模型

| 套餐 | 价格 | 核心功能 | 目标用户 |
|------|------|---------|---------|
| **Free** | $0 | 基础市场数据、5 个提醒、20 个收藏 | 新手、试用用户 |
| **Pro** | $39/月 或 $390/年 | 无限提醒、AI 分析、投资组合、高级图表 | 活跃交易者 |
| **Elite** | $99/月 或 $990/年 | 所有 Pro 功能 + API 访问 + 专属支持 + 早期功能 | 专业投资者/机构 |

#### 加密货币支付支持

**支持的加密货币**:
- USDT (TRC20/ERC20) - 推荐，稳定币
- USDC (ERC20) - 推荐，稳定币
- BTC - 比特币
- ETH - 以太坊
- SOL - Solana
- BNB - 币安币

**支付集成方案**:
1. **Coinbase Commerce** - 最简单，支持多种加密货币
2. **BitPay** - 成熟稳定，费率约 1%
3. **NOWPayments** - 费率 0.5%，支持自动兑换法币
4. **自研钱包集成** - 成本最高，但可控

**推荐方案**: Coinbase Commerce + NOWPayments 双备份

**加密支付激励**:
- 使用加密货币支付享受 **9 折优惠**
- 年付 + 加密货币支付 = **8 折** ($312/年 或 $792/年)

## 📈 转化优化

### 1. 转化漏斗设计

```
访客 (100%)
  ↓ 60%
注册免费用户 (60%)
  ↓ 10%
活跃用户 (6%)
  ↓ 8%
付费用户 (0.48% of 访客)
```

**优化目标**:
- 访客→注册：60% → 70%
- 注册→活跃：10% → 15%
- 活跃→付费：8% → 10%
- **整体转化**: 0.48% → 1.05%

### 2. 付费墙策略

**免费版限制**（促使用户升级）:
| 功能 | 免费版 | Pro 版 | Elite 版 |

| 收藏货币 | 20 个 | 无限 | 无限 |
| AI 分析 | ❌ | ✅ | ✅ |
| 投资组合 | ❌ | ✅ (基础) | ✅ (高级) |

# 主页功能说明

## 页面结构

主页现在包含以下几个主要部分：

### 1. Hero Section (英雄区)
- 应用标题和介绍
- CTA 按钮
- 吸引用户注意力

### 2. Market Overview (市场概览)

#### 市场统计卡片
- **Total Market Cap** - 加密货币总市值
- **24h Volume** - 24小时总交易量
- **BTC Dominance** - 比特币市场占有率
- **ETH Dominance** - 以太坊市场占有率

#### Top Gainers (涨幅榜)
- 显示 24 小时内涨幅最大的前 6 名货币
- 包含价格、涨跌幅、市值
- 快速收藏功能

#### Top Losers (跌幅榜)
- 显示 24 小时内跌幅最大的前 6 名货币
- 帮助用户发现投资机会或风险

#### Trending (热门货币)
- 按交易量排序的热门货币
- 前 6 名最活跃的交易货币

### 3. All Cryptocurrencies (全部货币列表) 🆕

#### 搜索功能
- 按名称或符号搜索
- 实时过滤结果
- 自动重置到第一页

#### 完整数据表格
显示的列：
- **排名** - 按市值排序的全球排名
- **名称/符号** - 货币完整名称和交易符号
- **价格** - 当前实时价格（美元）
- **24h %** - 24小时价格变化百分比
- **7d %** - 7天价格变化百分比
- **市值** - 总市值（缩写格式：T/B/M）
- **交易量** - 24小时交易量（缩写格式）
- **操作** - 快速收藏按钮

#### 分页浏览
- 每页显示 20 条记录
- 上一页/下一页按钮
- 快速跳转到指定页码
- 显示当前浏览范围

#### 数据刷新
- 手动刷新按钮
- 保持当前页码和搜索条件

## 数据格式说明

### 价格格式
- 小于 $1: `$0.000123` (6位小数)
- 大于 $1: `$45,000.50` (2位小数)
- 包含千位分隔符

### 大数字格式
- 万亿: `$1.25T`
- 十亿: `$850.30B`
- 百万: `$45.60M`

### 百分比格式
- 正数: `+2.50%` (绿色 ↗)
- 负数: `-1.25%` (红色 ↘)
- 保留 2 位小数

## 用户操作

### 浏览货币
1. 打开主页
2. 向下滚动到 "All Cryptocurrencies" 部分
3. 查看第一页的 20 条数据

### 搜索货币
1. 在搜索框输入关键词（如 "bitcoin" 或 "BTC"）
2. 系统自动过滤显示匹配结果
3. 点击刷新可以重新加载

### 翻页浏览
1. 点击 "Next" 查看下一页
2. 点击 "Previous" 返回上一页
3. 点击页码直接跳转

### 添加收藏
1. 找到想收藏的货币
2. 点击该行最右边的收藏按钮（心形图标）
3. 图标变为实心表示已收藏
4. 再次点击可取消收藏

### 查看价格趋势
- 绿色 ↗ 表示价格上涨
- 红色 ↘ 表示价格下跌
- 查看 24h % 和 7d % 了解短期和中期趋势

## API 接口

### 市场概览
```
GET /api/v1/currency/market-overview
```

### 货币列表
```
GET /api/v1/currency/list?page=1&page_size=20&keyword=bitcoin
```

参数：
- `page`: 页码（从 1 开始）
- `page_size`: 每页数量（默认 20）
- `keyword`: 搜索关键词（可选）

## 快速开始

1. **查看市场总览**
   - 打开主页即可看到市场统计
   - 快速了解市场整体情况

2. **发现热门货币**
   - 查看 Top Gainers 找到上涨机会
   - 查看 Top Losers 了解市场风险
   - 查看 Trending 了解交易热度

3. **浏览所有货币**
   - 滚动到 All Cryptocurrencies 部分
   - 使用搜索快速找到目标货币
   - 通过分页浏览更多选项

4. **管理收藏**
   - 点击心形图标添加收藏
   - 在 Profile 页面查看所有收藏
   - 快速访问关注的货币

## 常见问题

**Q: 数据多久更新一次？**
A: 数据会定期从后端同步，可以点击刷新按钮手动更新。

**Q: 如何查看更多货币？**
A: 使用页面底部的分页控件翻页浏览。

**Q: 搜索支持哪些内容？**
A: 支持按货币名称（如 Bitcoin）或符号（如 BTC）搜索。

**Q: 收藏后在哪里查看？**
A: 登录后在 Profile 页面的 Favorites 标签可以查看所有收藏。

**Q: 如何设置价格提醒？**
A: 需要登录后在 Profile 页面创建 Alert。
